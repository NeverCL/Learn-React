## 准备

淘宝源：`npm config set registry https://registry.npm.taobao.org`

升级npm：`npm i npm -g`

express：`npm i express -S`

安装webpack：`npm i -D webpack webpack-cli webpack-dev-server`(不建议全局安装)

运行webpack：`npx webpack`(Node 8.2+ 版本提供的`npx`命令)
